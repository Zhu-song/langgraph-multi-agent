# 5分钟快速上手

本文档帮助你在 5 分钟内快速体验 LangGraph 多智能体助手的核心功能。

## 前置条件

- 已完成 [安装](./installation.md) 和 [配置](./configuration.md)
- 服务已成功启动（`./start.sh`）

## 步骤 1：访问系统

打开浏览器，访问 http://localhost:3000

你应该能看到登录页面。

## 步骤 2：注册账号

1. 点击 "注册账号"
2. 输入用户名和密码
3. 点击 "注册"

> 密码会使用 SHA256 加密存储，请妥善保管。

## 步骤 3：创建对话

登录后，点击左侧 "新建对话" 按钮：

1. 输入对话标题（如 "测试对话"）
2. 选择对话模式：
   - **智能对话**：标准 AI 对话，支持工具调用
   - **流式对话**：实时流式输出响应
   - **RAG 问答**：基于知识库回答
   - **RAG 流式**：知识库问答 + 流式输出
   - **真实 RAG 流式**：逐字流式输出知识库结果

3. 点击 "创建"

## 步骤 4：体验基础对话

在输入框中输入消息，体验 AI 对话：

```
你好，请介绍一下你自己
```

AI 会介绍自己是 LangGraph 多智能体助手，并说明支持的 15 种工具。

## 步骤 5：体验工具调用

### 5.1 数学计算

```
计算：1234 * 5678 + 999
```

AI 会自动调用 `calculator` 工具进行计算。

### 5.2 联网搜索

```
搜索今天的天气
```

> 注意：联网搜索是高危工具，会触发人工审核。

如果启用了人工审核，你会看到审核提示，需要在审核面板中批准后才能执行。

### 5.3 中英翻译

```
翻译：Hello, how are you?
```

AI 会调用 `translate` 工具进行翻译。

## 步骤 6：使用知识库（RAG）

### 6.1 上传文档

1. 点击左侧 "知识库" 标签
2. 点击 "上传文档"
3. 选择要上传的文件（支持 TXT, MD, PDF）
4. 点击 "确认上传"

### 6.2 更新知识库

上传文档后，点击 "更新知识库" 按钮，系统会：
1. 加载文档
2. 分块处理
3. 向量化存储到 ChromaDB

### 6.3 RAG 问答

1. 创建一个 "RAG 问答" 模式的对话
2. 提问与上传文档相关的问题

```
根据知识库，总结一下文档的主要内容
```

AI 会基于知识库内容回答，并标注信息来源。

## 步骤 7：体验人工审核

### 7.1 触发高危工具

在对话中请求执行高危操作：

```
搜索最新的人工智能新闻
```

`web_search` 是高危工具，会触发人工审核。

### 7.2 处理审核

1. 点击左侧 "审核" 标签
2. 查看待审核的工具调用请求
3. 选择 "通过" 或 "拒绝"
4. 如果选择通过，AI 会继续执行搜索并返回结果

## 步骤 8：管理多对话

### 8.1 创建多个对话

点击 "新建对话" 创建多个对话，体验多会话管理：
- 对话 1：日常问答
- 对话 2：知识库问答
- 对话 3：代码辅助

### 8.2 切换对话

点击左侧对话列表中的对话标题即可切换。

### 8.3 置顶/删除对话

鼠标悬停在对话上，会出现置顶和删除按钮。

## 核心功能一览

| 功能 | 说明 | 体验方式 |
|------|------|----------|
| 智能对话 | 支持工具调用的 AI 对话 | 直接输入问题 |
| 流式输出 | 实时显示 AI 思考过程 | 选择流式对话模式 |
| RAG 检索 | 基于私有知识库回答 | 上传文档后提问 |
| 知识图谱 | 实体关系推理 | 配置 Neo4j 后使用 |
| 人工审核 | 高危工具拦截 | 触发 web_search |
| 多用户 | 数据隔离 | 注册多个账号测试 |
| 多对话 | 会话管理 | 创建多个对话 |

## 常见问题

### Q: AI 不调用工具？

确保使用 "智能对话" 或 "流式对话" 模式，并且问题明确需要工具帮助。

### Q: RAG 回答不准确？

- 确保文档已成功上传到知识库
- 点击 "更新知识库" 确保文档已索引
- 尝试调整提问方式

### Q: 人工审核不弹出？

检查 `.env` 中 `ENABLE_APPROVAL=true` 是否配置正确。
